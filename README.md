### Lab02 
In this lab, you will design and build a frequency-selective filter that allows a desired signal to pass
through while rejecting interfering signals. Specifically, we will design and build an “electric eye” light
beam-interrupter safety system. The system consists of a transmitter that projects a beam of light onto a
receiver (see Figure 1). When an object physically obstructs the beam, a small buzzer sounds (or a
warning light is powered on). When the obstructing object is removed, the buzzer stops.
